'Very'

Adverb 'very' is one exceptional example as it is used to amplify effect of the verb or add tension to it. It can carry tone difference too. So there is good amount of adverbs that can carry this amplification.

Adverb Translation Usage Description Formality Emphasize/Strength 例
迚とても very; awfully; exceedingly とても is the most common adverb for 'very'. Neutral
とても楽しい。
大変たいへん very; greatly; terribly; awfully たいへん is also used to say 'very'. It also covers meanings greatly, terribly, awfully
It differs from とても as it sounds more formal and has more seriousness because it is used for 'serious or difficult situations'. 		Neutral
大変忙しい。
非常ひじょうに extremely; very This one is directly translated as 'Extremely'. But it can be used as 'very' too but is more formal and quite more serious. Formal / Neutral

非常に危険です。
非常に危ない。

凄すごく awfully; very; extremely; immensely; really It is used either for most positive or negative extent. In formal speech, it can be replaced by 非常に. Casual / Neutral
凄く楽しい。
