Combined Particles

Combined particles (e.g. には, では, への, にも, ...) are, of course, particle compounds that have more than one particle. They are mostly used for forming complex ideas, emphasizing, precision, manner, etc.
In order to master them, one things is crucial: knowing how different particles work.

  • には
  • では
  • へは
  • にも
  • でも
  • へも
  • での
  • への
  • のに
  • ので

には

「には」 is particle combination that involves two particles: 「に」 and 「は」. It's primary function is self-explanatory - to emphasize target of the sentence.

Emphasizing Target / Location:

例
東京とうきょうには、美味おいしいラーメン屋やが沢山たくさんある。
As for Tokyo, there are many taste ramen shops.
真理子さんの家の屋根には、赤い葉が多い。
ON Mariko's house, there are many red leaves.

Turning Location (に) into the Topic (は):

例
この箱の中には、何もない。
(As for) This box has nothing inside.
(Maybe other things have?)

では

「では」 also covers the same mechanism as NI WA but instead of NI it uses particle 「で」. It means that function of で particle is present in this combination.
Alas, if you thought that was simple as that, there are bad news. There is other case where では can appear and confuse you since it looks the same as this primary combination.

例
Instrumental:
車では、簡単かんたんに行ける。
It is easier if you go by car.

Place of Action:
公園こうえんの池では、泳およいじゃダメよ!
You mustn't swim in park's pond, you know.

Example info: There may be other methods, but car simplifies it and DE has the function of instrumental.

ては / では Conjunction

If this combination is used as conjunction, it means "then, well then...". It, then, gets a little tricky as there are some different variants and rules for this conjunction:

  • ては

    • て form + は
    • 例
      あの鶏肉とりにくを食べては、ダメよ。あれは上司じょうしの昼ご飯だよ。
      You should not eat that chicken! That's boss's lunch!
    It follows the same rules as て form - て "suffix" is either voiced or not and is attached to euphonic (音便形) base.

  • では

    • て form (で) + は
    • 例
      この質問しつもんを選えらんでは、ダメよ。なぜなら、正しくないからだよ。
      You should not choose that answer because it is not correct.

    It follows the same rules as て form - て "suffix" is either voiced or not and is attached to euphonic (音便形) base.
    In this case て it is voiced and gets dakuten ("). If you wonder why, well, that is because euphonic base works that way and you just have to memorise it for whole hiragana う column.

    - To simplify this, verb ending syllables that demand voiced て particle suffix are: (ぐ, ぬ, ぶ, む). Rest (う, く, つ, る) gets normal て.

    Important!

    In casual speech and spoken Japanese generally, there is one important thing to note:

    • ては　⇒　ちゃ (cha)
      • 例
        バカ！その手紙を書いてはダメよ。
        You fool! You must not write that letter.
        バカ！その手紙を書いちゃダメよ。
        You fool! You must not write that letter.
    • では　⇒　じゃ (ja)
      • 例
        ダムの湖では、泳いではいけないよ。
        You must not swim in dam lake.
        ダムの湖では、泳よいじゃいけないよ。
        You must not swim in dam lake.

へは

This one acts the same as 「には」 but uses へ particle for direction of movement. There is more to be said.

にも

Well, this one is new kind of combinations because it uses another kind of contextual particles, particle 「も」. As particle も primarily behaves as set inclusion particle, it has exact same function there, but has also に as well. We can divide its function in two parts:

  • 「にも」 as particle where に has function of Place of EXISTENCE / ARRIVAL, DESTINATION or TIME.
  • 「にも」 as particle where に has function of Target of Action (indirect object).

If に functions as first case, then, it includes manner marked by に in the set of things mentioned and gets close to what is in English said as "even".
As for the second usage, it is very similar. But it has manner of emphasis and disappointing tone or somehow surprised tone based on context.

例
僕ぼくの寝屋ねやにも静しずけさがないよ。
There isn't peace even in my own bedroom!
どうしたの。その子供こどもにも分わかってるよ。
What's the matter? Even the kid understands!

のに

This combination has two main cases:
  • Nominalization + に (in function of either indirect object or place of existence / arrival)
    • 例
      お金を盗むのに幸せがない。
      There is no luck in stealing money.
  • Conjunctive のに which means "although, despite, even though, and yet"
    • 例
      彼は足が痛かったのに、試合に勝った。
      Even though his leg hurt, he won the match.
      曇になったのに、自転車で山に回った。
      Even though it got cloudy, we went around the mountain by bike.
